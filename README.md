## Manual de Usuario

**Importante:** el usuario previo a la compilacion debe instalar:
```sh
sudo apt install qt6-base-dev qt6-multimedia-dev
sudo apt-get install libyaml-cpp-dev
```

__Compilacion:__
```sh
make
```
__Instalacion:__
```sh
make install
```
__Borrar Ejecutables:__
```sh
make clean
```
